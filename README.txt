Adds a Customizable Confirmation Alert sitewide or your post/page using shortcode.

== Description ==

Totally New and Customizable Popup

Multipurpose Confirmation Alert can be use to  show confirmation alerts on your website. It can also be add to a separate page/post. It allows you to customize the Alert Box Title, Description and even the buttons. It has predefined themes and backgrounds along with other features. 

== Installation ==
1. Upload and extract `adult-confirmation.zip` to the `/wp-content/plugins/` directory 
OR
1. Upload `adult-confirmation.zip` to the Plugins > Add New > Upload

2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WP-Admin -> Multipurpose Alert -> Settings for sitewide enable or Add shortcode [acxp_malert] to any page/post. 

== Frequently Asked Questions ==
- How can i use the plugin?
Ans: Look for Documentation in WP-Admin -> Multipurpose Alert -> How to Use?

- How can i use Adult Confirmation on whole website?
Ans: Go to WP-Admin -> Multipurpose Alert -> Settings for sitewide enable
